# Промпты для ИИ-дизайнера для 3 feature-блоков

Ниже 3 отдельных промпта для генерации изображений для блоков лендинга `01 / 02 / 03`.

Общие требования для всех изображений:
- Стиль должен совпадать с текущим сайтом.
- Современный минималистичный визуал.
- Чистая коммерческая эстетика.
- Премиальный, но сдержанный стиль.
- Светлый фон или очень светлая сцена.
- Без кислотных цветов.
- Без синего фона.
- Без перегруженной инфографики.
- Без лишнего текста внутри изображения.
- Без интерфейсов, которые выглядят как дешевые стоковые макеты.
- Без мультяшности.
- Без 3D-иконок в стиле маркетплейс-шаблонов.
- Изображение должно хорошо смотреться рядом с крупным черным заголовком и большим количеством белого пространства.
- Визуал должен быть лаконичным, дорогим, современным и соответствовать b2b-лендингу.
- Формат изображения: горизонтальный.
- Соотношение сторон: `4:3`.
- Рекомендуемый размер: `1600 x 1200 px`.
- Важные объекты размещать ближе к центру, чтобы ничего не обрезалось на адаптиве.

## Промпт 1 — Блок «Быстрые лиды»

**Задача:**
Создай изображение для блока лендинга Avito Ads в стиле современного минималистичного b2b-сайта. Изображение должно визуально передавать идею быстрого получения заявок, дешевых кликов и быстрого запуска рекламной кампании.

**Что должно быть в кадре:**
- ассоциация с быстрым стартом рекламной кампании;
- ощущение движения, скорости, первых результатов;
- можно показать ноутбук, экран аналитики, рекламный кабинет, поток входящих заявок или абстрактную бизнес-сцену;
- допускаются элементы digital-маркетинга, но без перегруза интерфейсами;
- визуал должен показывать, что результат приходит быстро и без лишней сложности.

**Стиль:**
- минимализм;
- светлая палитра;
- премиальный и современный визуальный язык;
- аккуратные нейтральные оттенки;
- можно добавить мягкие акценты в фирменной логике Avito Ads, но очень сдержанно;
- изображение должно смотреться дорого и технологично.

**Композиция:**
- главный объект или сцена по центру;
- справа и по краям можно оставлять воздух;
- изображение должно хорошо читаться в блоке сайта рядом с большим текстом слева;
- избегать мелких деталей, которые потеряются при уменьшении.

**Размер:**
- `1600 x 1200 px`
- соотношение `4:3`

**Готовый промпт для генерации:**

> Создай горизонтальное изображение 1600x1200 в соотношении 4:3 для premium b2b-лендинга про Avito Ads. Сюжет должен передавать идею быстрых лидов, дешевых кликов и быстрого запуска рекламной кампании. Покажи современную минималистичную digital-сцену: можно использовать ноутбук, рекламный кабинет, входящие заявки, аналитику или абстрактный образ быстрого результата. Стиль чистый, светлый, дорогой, современный, технологичный, без визуального шума. Много воздуха, аккуратная композиция, без синего фона, без кислотных цветов, без лишнего текста, без мультяшности, без дешевого стокового ощущения. Изображение должно органично сочетаться с черно-белым минималистичным лендингом и выглядеть как часть премиального сайта агентства.
